# References de datasets open source - Assurance auto et expertise dommages

Date de recherche : 2026-05-07

Objectif : identifier des jeux de donnees utilisables pour un projet d'assurance automobile autour de deux besoins :

- rapports d'expert / donnees proches d'un rapport d'expertise ;
- photos d'accidents ou de vehicules endommages annotees avec les dommages.



## Datasets de photos de vehicules endommages annotees

| Dataset | Lien | Type / annotations | Licence / acces | Description et interet |
|---|---|---|---|---|
| CarDD - Car Damage Detection | https://cardd-ustc.github.io/ | 4 000 images haute resolution, plus de 9 000 instances annotees, categories : dent, scratch, crack, glass shatter, lamp broken, tire flat. Taches : classification, detection, segmentation d'instance, salient object detection. | Acces via formulaire de licence ; code et papier disponibles. | Reference academique majeure pour l'assurance auto. Tres pertinent pour detecter et segmenter les dommages exterieurs a partir de photos de sinistre. |
| CarDD sur Hugging Face | https://huggingface.co/datasets/sashankrudra/CarDD | Repackaging du dataset CarDD en format image/object detection. | Verifier les conditions de licence avec la source originale CarDD avant usage commercial. | Plus simple a explorer techniquement que le formulaire original, mais il faut respecter la licence originale. Utile pour prototypage rapide. |
| VehiDE - Automatic Vehicle Damage Detection | https://www.kaggle.com/datasets/hendrichscullen/vehide-dataset-automatic-vehicle-damage-detection | 13 945 photos haute resolution, plus de 32 000 occurrences de dommages annotees ; detection et segmentation. | Kaggle, Apache 2.0 indique sur la page. | Tres bon candidat pour un moteur d'analyse visuelle de sinistres. Volume nettement superieur a CarDD, oriente explicitement car insurance. |
| Humans in the Loop - Car Parts and Car Damages Dataset | https://humansintheloop.org/resources/datasets/car-parts-and-car-damages-dataset/ | 1 812 images : 998 images de pieces auto et 814 images de dommages ; 24 851 polygones. Classes dommages : dent, cracked, scratch, flaking, broken part, paint chip, missing part, corrosion. | CC0 1.0 Public Domain ; acces via formulaire. | Tres utile pour relier dommage et piece touchee. Bon dataset pour segmentation fine et annotation polygonale. |
| COCO Car Damage Detection Dataset | https://www.kaggle.com/datasets/lplenka/coco-car-damage-detection-dataset | 80 images, annotations COCO ; dommages et pieces touchees : headlamp, front bumper, hood, door, rear bumper. | CC0 Public Domain selon Kaggle. | Petit dataset mais propre pour tests rapides, validation de pipeline COCO, exemples de segmentation et bounding boxes. |
| Annotated Dataset of Car Parts with Damage | https://www.kaggle.com/datasets/shubhammkumaar/car-damage-parts-detection | Images de vehicules avec annotations JSON polygonales ; dommages : scratches, dents, broken parts ; pieces : bumper, door, side mirror, fender, etc. | MIT selon Kaggle. | Interet fort pour une logique "rapport d'expert" car il annote simultanement la piece et le type de dommage. |
| Car Accidents and Deformation Dataset annotated | https://www.kaggle.com/datasets/marslanarshad/car-accidents-and-deformation-datasetannotated | Images accident/non-accident, labels YOLO avec classes de severite/deformation : no accident, minor, moderate, severe, totaled. | CC0 Public Domain selon Kaggle. | Utile pour estimer une severite globale et detecter les cas epave / vehicule tres endommage. Moins fin que les datasets de segmentation de dommages. |
| CarDD with YOLO Annotations | https://www.kaggle.com/datasets/gabrielfcarvalho/cardd-with-yolo-annotations-images-labels | CarDD converti en labels YOLO : dent, scratch, crack, glass shatter, lamp broken, tire flat. | CC BY-NC 4.0 selon Kaggle ; non commercial. | Pratique si le modele cible est YOLO. A utiliser avec prudence pour un projet commercial a cause de la clause non commerciale. |
| SYNDCAR - Synthetic Vehicle Damage Detection and Parts Segmentation | https://data.mendeley.com/datasets/hzpj48krdt/1 | 245 images haute resolution synthetiques ; dommages : broken glass, broken lights, cracks, scratches ; pieces vehicule annotees en segmentation. Formats YOLO detection et segmentation. | CC BY 4.0. | Dataset recent, documente, utile pour augmenter les classes rares ou tester un pipeline sans probleme de confidentialite. Limite : donnees synthetiques, donc domain gap avec photos reelles. |
| Trucks Damage Dataset | https://huggingface.co/datasets/salim4n/trucks-damage-dataset | 795 images de camions, annotations COCO avec masques de segmentation : scratch, dent, broken part, missing part, crack, lamp broken, etc. | Licence a verifier : la page demande de specifier la licence. | Moins centre voiture particuliere, mais interessant pour flotte, utilitaires et assurance professionnelle. Ne pas utiliser en production sans clarification de licence. |

## Datasets proches d'un rapport d'expertise ou d'un dossier de sinistre

| Dataset | Lien | Type / champs | Licence / acces | Description et interet |
|---|---|---|---|---|
| Vehicle Insurance Claim Fraud Detection | https://www.kaggle.com/datasets/shivamb/vehicle-claim-fraud-detection | Donnees structurees de sinistres auto : attributs vehicule, details d'accident, police, cible fraude `FraudFound_P`. | CC0 Public Domain selon Kaggle. | Bon dataset pour la partie "analyse de coherence / fraude" d'un dossier sinistre. Ne contient pas de rapport d'expert texte ni photos. |
| Car Insurance Fraud Detection Dataset | https://www.kaggle.com/datasets/ahluwaliasaksham/car-insurance-fraud-detection-dataset | Donnees structurees : incident type, collision type, incident severity, police report available, claim amount, total claim amount, fraud reported. | MIT selon Kaggle. | Interessant pour simuler les champs d'un dossier de sinistre et entrainer un modele de risque/fraude. A verifier sur la provenance, car la page le presente comme "realistic ML-ready". |
| Auto Insurance Claims Data | https://www.kaggle.com/datasets/buntyshah/auto-insurance-claims-data | CSV de reclamations assurance auto, 40 colonnes. | Licence inconnue selon Kaggle. | Dataset tres utilise pour analyses de sinistres, mais la licence inconnue impose de le limiter a l'exploration ou de demander clarification. |
| FreMTPL - French Motor TPL Insurance Claims Data | https://baselight.app/u/kaggle/dataset/karansarpal_fremtpl_french_motor_tpl_insurance_claims | Donnees actuarielles francaises : caracteristiques de risque, nombre de sinistres, montants de sinistres ; deux tables freq/severity reliees par policy ID. | GPL-2.0 indique sur Baselight/Kaggle mirror. | Tres utile pour modelisation actuarielle en assurance auto francaise : frequence, severite, cout. Ne decrit pas les dommages visuels. |
| usautoBI - Automobile Bodily Injury Claim Dataset | https://dutangc.github.io/CASdatasets/reference/usautoBI.html | 1 340 reclamations corporelles auto ; age, sexe, avocat, ceinture, assurance, perte economique. | Dataset R package CASdatasets ; verifier la licence du package/source. | Pertinent pour la dimension indemnisation / blessure / cout, mais pas pour l'expertise carrosserie. |
| Insurance Claims Data - insuranceData / CAS datasets mirror | https://www.kaggle.com/datasets/saisatish09/insuranceclaimsdata | Regroupe AutoBI, AutoClaims, AutoCollision : pertes, montants payes, severite moyenne, nombre de sinistres. | Licence inconnue selon Kaggle. | Utile pour benchmark statistique et prediction de couts. A eviter pour usage commercial sans licence claire. |
| BAAC / ONISR - Accidents corporels de la circulation France | https://www.onisr.securite-routiere.gouv.fr/en/data-tools/open-data | Donnees officielles d'accidents corporels : caracteristiques, lieux, vehicules, usagers ; fichiers CSV par annee. | Open data public via data.gouv.fr. | Excellent pour enrichir le contexte accident : date, lieu, type de route, vehicules impliques, gravite. Ne contient ni photos ni couts de reparation. |
| NCID Private Motor Insurance Data - Ultimate Claims | https://data.gov.ie/dataset/ncid-part-2-claims-ultimate-claims | Donnees agregees de sinistres moteur prive, par annee d'accident, cout/nombre ultime de sinistres. | Creative Commons Attribution 4.0. | Utile pour cadrer des analyses macro de couts et tendance sinistres, pas pour entrainer un modele individuel d'expertise. |
| Vehicle Claim - Synthetic dataset | https://www.juheapi.com/datasets/vehicle-claim | Donnees synthetiques pour audit de claims : marque, modele, kilometrage, type de dommage, dommage specifique, complexite, heures et cout de reparation. | Presente comme open source community ; verifier le depot GitHub lie avant usage. | Tres proche de la structure d'un rapport d'estimation/reparation. Bon pour prototyper un schema de donnees, mais ne remplace pas des donnees reelles. |
| Auto Insurance Claims - Zenodo | https://zenodo.org/records/13381118 | Fichier Excel de donnees de claims auto / fraude. | Dataset ouvert sur Zenodo ; verifier la licence affichee dans Zenodo avant usage. | Alternative aux datasets Kaggle pour experimentation. Provenance et schema a auditer avant integration. |
